## 🔬 Featured Open-Source Project

### [Enterprise Tabular RAG Analytics Engine](https://github.com/bogdandarabanul/spreadsheet-rag-engine) 🚀
An asynchronous Python/FastAPI backend designed to ingest and query complex financial spreadsheets using natural language.
* **Advanced RAG:** Implements a dual-pass `openpyxl` parser and utilizes PostgreSQL (`pgvector`) for 768-dimensional semantic embedding retrieval.
* **Deterministic AI Execution:** Integrates `gemini-2.5-flash` using strict XML tagging and Chain-of-Thought (CoT) prompting to enforce step-by-step logical reasoning.
* **Financial Parity:** Enforces absolute calculation correctness using Python’s `decimal` module to eliminate floating-point anomalies.
* **Enterprise Practices:** 100% offline-runnable test coverage via `pytest-asyncio` with API mocking, fully containerized via Docker and automated through GitHub Actions.
